# api_province_vn
Api tất cả các tỉnh thành Việt Nam

#Chức năng: 
  - Đây là api tổng hợp tất cả các (Tỉnh/Thành/Quận/Huyện/Phường/Xã)
  - Dự liệu được cập nhập mới nhất vào ngày 19/03/2024 tại https://danhmuchanhchinh.gso.gov.vn/
